## About BIO-Connect

BIO-Connect is a web application with Laravel8 framework

## Install BIO-Connect

-   clone
-   composer install
-   cp .env.example to .env
-   php artisan key:generate
-   php artisan jwt:secret
-   chmod -R 755 storage bootstrap/cache
-   edit and adjust .env
-   create database
-   php artisan migrate
-   to excute http://bio-connect/api
